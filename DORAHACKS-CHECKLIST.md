# Fast Submission Checklist

1. Create a GitHub repo named `mantle-agent-sentinel`.
2. Upload everything in this folder.
3. Enable GitHub Pages from Settings -> Pages -> Deploy from a branch -> main / root.
4. Record a 2 minute screen video using `video-script-ja.md`.
5. Paste `submission-en.md` into DoraHacks.
6. Use `AI DevTools` as the primary track.
7. If you have time, deploy `contracts/AgentDecisionLog.sol` to Mantle Sepolia and replace `TBD` with the contract address.

Minimum viable submission:

- Repo link
- Live demo link or uploaded video
- Project description from `submission-en.md`
- Track: AI DevTools

Stretch if time remains:

- Mantle Sepolia deployment
- Contract explorer link
- Short README screenshot or demo GIF
